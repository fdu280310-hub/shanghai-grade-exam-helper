# 题目改编规则

将外地（全国卷、北京卷、江苏卷等）题目改编为上海等级考风格的通用规则。

## 改编原则

### 1. 情景化命题（核心特色）

上海等级考的最大特色是**每个大题目都有真实情境引入**，而非直接出题。

改编要点：
- 为每个大题目添加1-2段情景引入文字
- 情境必须真实、科学、与题目内容自然衔接
- 可以是：工业流程、科研发现、环保案例、生活应用、新材料技术
- 情境不是装饰，后续子题应能从情境中找到线索
- 避免："小明在实验室……""某同学做实验……"等低幼化表述
- 推荐："某化工厂采用……""航天器中……""最新研究发现……"

### 2. 题型配比

上海等级考典型的大题结构：

```
大题目（含情景引入）
├── 第1小题：选择题（单选或不定期），考查基础概念理解
├── 第2小题：选择题或填空题，考查情境相关知识点
├── 第3小题：填空题或简答题，考查推理/分析能力
└── 第4小题：计算题或开放性简答，考查综合能力

难度递进：简单 → 中等 → 较难
```

### 3. 选择题规则

**单选题**：
- 4个选项（A/B/C/D）
- 每题1个正确选项
- 不特别标注

**不定项选择题**：
- 4个选项（A/B/C/D）
- 每题1~2个正确选项
- 在题干中标注"（不定项）"
- 评分：1个正确选项的多选不给分；2个正确选项的漏选1个得一半分，错选不得分

### 4. 子题编号

```
大题：一、二、三、四、……
子题：1．2．3．4．……（阿拉伯数字 + 全角点）
选项：A．B．C．D．（大写字母 + 全角点）
```

### 5. 难度适配

上海等级考 vs 全国卷的特点对比：

| 维度 | 全国卷 | 上海等级考 |
|------|--------|-----------|
| 计算量 | 较大 | 适中 |
| 推理深度 | 中等 | 较深 |
| 情境要求 | 可选 | 必须 |
| 开放性问题 | 少 | 较多 |
| 不定项选择 | 无 | 有 |
| 文字表述 | 简洁 | 较详细 |

改编时需注意：
- 减少纯粹的计算量（不必每个题都算出精确数值）
- 增加"说明理由""简述原因"类问题
- 将抽象概念题嵌入具体情境
- 适当增加文字表述而不是简化

### 6. 不改变的内容

以下内容在改编时尽量保持原样：
- 核心化学/物理原理
- 数值答案的正确选项
- 原题考查的知识点
- 题目难度不应大幅偏离原题

## 改编步骤示例

### 原题（全国卷风格）

> 已知反应 2SO₂(g) + O₂(g) ⇌ 2SO₃(g) ΔH = -196.6 kJ·mol⁻¹。
> (1) 计算该温度下的平衡常数。
> (2) 若增大压强，平衡向哪个方向移动？

### 改编后（上海等级考风格）

> **一、硫酸工业中的化学平衡**
>
> 硫酸是化学工业中的重要产品。工业上采用接触法制硫酸，其核心反应为 SO₂ 的催化氧化：
> 2SO₂(g) + O₂(g) ⇌ 2SO₃(g) ΔH = -196.6 kJ·mol⁻¹。
> 某硫酸厂在 400°C、常压下进行此反应，尾气中仍含有约 0.3% 的 SO₂。
>
> 1．（不定项）下列措施中能提高 SO₂ 平衡转化率的是__________。
> A．升高温度
> B．增大压强
> C．使用 V₂O₅ 催化剂
> D．增大 O₂ 的浓度
>
> 2．该温度下反应的平衡常数为 K。若保持温度不变，将容器体积压缩为原来的一半，此时 Q 与 K 的关系为 Q_______K（填"＞""＜"或"＝"）。
>
> 3．工业上通常不在极高压强下进行此反应，请结合化学反应原理和实际生产，简述理由。
