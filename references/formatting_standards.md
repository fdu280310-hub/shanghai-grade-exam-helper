# 上海等级考排版规范详解

本文档详细说明上海高中等级考（物理/化学）试卷的 Word 排版标准。

## 页面设置

| 参数 | 值 | twips |
|------|-----|-------|
| 纸张大小 | A4 (210mm × 297mm) | 11906 × 16838 |
| 上边距 | 2.54 cm | 1440 |
| 下边距 | 2.54 cm | 1440 |
| 左边距 | 3.18 cm | 1800 |
| 右边距 | 3.18 cm | 1800 |
| 页眉 | 1.5 cm | 851 |
| 页脚 | 1.75 cm | 992 |

## 字体层级体系

### 一级：试卷标题

```
字体：黑体 (SimHei)
字号：16pt (sz=32, szCs=32)
对齐：居中
行距：360 (1.5倍行距)
加粗：否
示例：高三化学练习
```

### 二级：副标题

```
字体：黑体 (SimHei)
字号：16pt (sz=32, szCs=32)
对齐：居中
行距：360
示例：高三教学资源包
```

### 三级：大题标题（一、二、三……）

```
字体：黑体 (SimHei)
字号：14pt (sz=28, szCs=28)
对齐：居中
行距：360
加粗：是
示例：一、铬及其化合物的工业应用
```

### 四级：小题题干

```
字体：宋体 (SimSun)
字号：10.5pt (sz=21, szCs=21)
对齐：左对齐
行距：360
加粗：题号部分加粗
缩进：悬挂缩进 420 twips
格式：数字后跟"．"（全角点），如"1．下列……"
不定项标注："（不定项）"紧跟在题号后
```

### 五级：选项 A/B/C/D

```
字体：宋体 (SimSun)
字号：10.5pt (sz=21)
对齐：左对齐
行距：360
制表位：4680 twips（两个选项之间的分界）
格式：单字母后跟"．"（全角点），如"A．pH＞7"
4个选项的布局：A  B 在第一行，C  D 在第二行
```

### 六级：情景引入正文

```
字体：宋体 (SimSun)
字号：10.5pt (sz=21)
对齐：左对齐（两端对齐亦可）
行距：293
缩进：首行缩进 480 twips（2个中文字符）
```

### 七级：考生注意

```
标题"考生注意："：黑体，10.5pt，加粗
条目文字：黑体，10.5pt
缩进：首行缩进 420 twips
条目编号：阿拉伯数字后跟"．"
```

### 八级：相对原子质量（化学专用）

```
字体：黑体，10.5pt
格式：中文"相对原子质量："后跟元素符号和数值
元素符号：Times New Roman
示例：H－1  O－16  S－32  Cr－52  Fe－56  Ba－137
分隔符：两个空格
```

## 默认运行属性

```
西文字体：Times New Roman
中文字体：宋体 (SimSun)
字号：10.5pt (sz=21, szCs=21)
语言：en-US (西文) / zh-CN (中文)
字距调整：kern w:val="2"
```

## 编号系统

### 大题编号 (abstractNum 1)

```
格式：chineseCountingThousand
文本：%1、（如：一、二、三、……十、十一、……）
```

### 子题编号（无预设 abstractNum，在 XML 中直接写入）

```
格式：decimal
文本：数字后跟全角点"．"，如"1．""2．"
```

## 特殊字符处理

### 化学式上标/下标

```xml
<!-- 下标 (H₂O 中的 2) -->
<w:vertAlign w:val="subscript"/>

<!-- 上标 (Ca²⁺ 中的 2+) -->
<w:vertAlign w:val="superscript"/>
```

### 可逆反应箭头

使用 Unicode 字符：⇌ (U+21CC) 或 ⇋ (U+21CB)

### 希腊字母

直接在 XML 中使用 Unicode。

## Tab 制表位（选项布局）

当使用制表位布局 A/B/C/D 选项时：

```
两列制表位：
<w:tabs>
  <w:tab w:val="center" w:pos="4680"/>
</w:tabs>
```

4个选项布局（A/B在第一行，C/D在第二行），使用居中对齐制表位使 AB 均衡分布。

## 兼容性设置

```
<w:compat>
  <w:compatSetting w:name="compatibilityMode" w:uri="http://schemas.microsoft.com/office/word"
    w:val="14"/>  <!-- Word 2010 兼容模式 -->
</w:compat>
```
