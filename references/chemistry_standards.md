# 化学科目规范

上海等级考化学科目的特殊格式和规范要求。

## 相对原子质量

每份化学试卷开头必须包含"相对原子质量"信息。

**必须根据每道题实际涉及的元素动态生成，不同试卷列出的元素不同！**

常用元素原子量参考值：
```
H－1    C－12   N－14   O－16   Na－23  Mg－24  Al－27  Si－28
P－31   S－32   Cl－35.5  K－39   Ca－40  Mn－55  Fe－56  Cu－64
Zn－65  Br－80  Ag－108  I－127  Ba－137
```

格式规范：
- 中文"相对原子质量："后直接跟元素符号和数值
- 元素符号用 Times New Roman
- 连接符用全角"－"
- 元素之间用两个空格分隔
- **关键：只列出题目中实际涉及的元素，不涉及的元素不要列。每题根据内容动态调整，不要照搬固定列表。**

## 化学式书写规范

### 上下标

```
下标：H₂O        → H<sub>2</sub>O
上标：Ca²⁺       → Ca<sup>2+</sup>
带电荷：SO₄²⁻    → SO<sub>4</sub><sup>2−</sup>
```

在 XML 中：
```xml
<w:r><w:rPr><w:vertAlign w:val="subscript"/></w:rPr><w:t>2</w:t></w:r>
```

### 化学方程式

格式：`反应物 → 生成物` 或 `反应物 ⇌ 生成物`（可逆）

状态符号：英文括号 (s), (l), (g), (aq)

示例：
```
2Na(s) + 2H₂O(l) → 2NaOH(aq) + H₂(g)↑
```

### 有机化学

- 碳链编号用下标数字
- 官能团名称用中文
- 结构简式用标准表示法（如 CH₃COOH）

## 常见情景主题

### 化工生产
- 侯氏制碱法（联合制碱法）
- 接触法制硫酸
- 氨的合成（哈伯法）
- 氯碱工业
- 石油裂解与分馏
- 硅酸盐工业（水泥、玻璃、陶瓷）

### 环境化学
- 废水处理中的氧化还原（Cr(VI)还原为Cr(III)）
- 烟气脱硫脱硝
- 碳捕集与封存 (CCS)
- 富营养化与水华
- 光化学烟雾
- PM2.5 与大气污染

### 材料化学
- 锂电池 / 钠离子电池
- 燃料电池（氢氧、甲醇）
- 光催化材料（TiO₂）
- 医用高分子
- 纳米材料
- 半导体材料（高纯硅制备）

### 分析化学
- 滴定分析（酸碱、氧化还原、沉淀）
- 比色法 / 分光光度法
- 色谱法
- pH 测定与应用
- 离子检验与鉴别

### 日常生活
- 食品添加剂与安全
- 药物合成与分析
- 水质检测
- 化妆品成分
- 家用清洁剂

## 不定项选择标注格式

化学不定项选择题在题干中标注"（不定项）"：

```
3．（不定项）下列离子在酸性溶液中能大量共存的是__________。
```

## 化学计量与有效数字

- 摩尔质量计算：保留1位小数（如 M=58.5 g·mol⁻¹）
- pH 值：保留2位小数
- 平衡常数：保留2-3位有效数字
- 浓度：保留3-4位有效数字
