# 物理科目规范

上海等级考物理科目的特殊格式和规范要求。

## 变量与符号规范

### 字体约定

| 类型 | 字体 | 示例 |
|------|------|------|
| 物理量（标量） | 斜体 Italic | *F*, *m*, *a*, *v*, *T* |
| 物理量（矢量） | 粗斜体 | **F**, **v**, **a** |
| 单位 | 正体 | N, kg, m/s², Pa |
| 常用常数 | 正体 | g, e, h, c |
| 数学函数 | 正体 | sin, cos, tan, ln, log |
| 希腊字母（变量） | 斜体 | *μ*, *θ*, *ω*, *α* |

### 常见符号表

| 符号 | 含义 | 书写 |
|------|------|------|
| *F* | 力 | 斜体大写 |
| *f* | 摩擦力 | 斜体小写 |
| *m* | 质量 | 斜体小写 |
| *a* | 加速度 | 斜体小写 |
| *g* | 重力加速度 | 斜体小写 |
| *v* | 速度 | 斜体小写 |
| *T* | 周期 | 斜体大写 |
| *E* | 能量/电场 | 斜体大写 |
| *B* | 磁感应强度 | 斜体大写 |
| *I* | 电流 | 斜体大写 |
| *U* | 电压 | 斜体大写 |
| *R* | 电阻 | 斜体大写 |
| *λ* | 波长 | 斜体 |

## 单位规范

- 使用国际单位制 (SI)
- 复合单位用"/"或负指数：m/s 或 m·s⁻¹
- 数字与单位之间留空格：10 m/s（中文语境下可省略）
- 常用单位：N, kg, m, s, Pa, J, W, V, A, Ω, T, Wb, Hz

## 公式排版

### 行内公式

简单公式写在行内：`F = ma`，`v = at`，`E = mc²`

### 单独成行的公式

复杂公式居中单独成行：
```
F = G·m₁m₂/r²
```
（在 docx XML 中使用居中对齐的段落）

### 上标/下标

```
上标：v², 10³, m·s⁻¹
下标：v₀, Eₖ, Fₙ
多层：v₀², Eₖ_max
```

## 数值规范

### 有效数字

- 一般计算保留 2-3 位有效数字
- 中间计算保留至少4位
- 最终结果按题面数据精度确定（通常2-3位）

### 科学记数法

- 使用 ×10ⁿ 格式：6.02 × 10²³
- 不使用 E 表示法

## 常见情景主题

### 交通与运输
- 磁悬浮列车原理（电磁感应、直线电机）
- 高速公路弯道设计（向心力、摩擦力）
- 电动汽车能量回收（能量转换、电磁感应）
- 高铁运行中的相对运动
- 交通安全距离与反应时间（匀变速运动）

### 航空航天
- 卫星轨道与变轨（万有引力、圆周运动）
- 火箭发射中的超重失重（牛顿定律）
- 空间站微重力环境
- 天宫号中的物理实验
- 火星探测器着陆（能量、缓冲）

### 体育运动
- 跳高/跳远力学分析（抛体运动）
- 游泳中的推进力（牛顿第三定律、流体阻力）
- 台球碰撞（动量守恒、弹性碰撞）
- 蹦极/蹦床（能量转换、简谐运动）
- 短跑起跑（摩擦力、加速度）

### 医疗技术
- CT/MRI 物理原理（X射线、核磁共振）
- 超声波诊断（声波、多普勒效应）
- 放射治疗（放射性衰变、半衰期）
- 心脏起搏器（电路、电容）
- 光学内窥镜（全反射、光纤）

### 现代科技
- 5G通信（电磁波、波长频率）
- 芯片制造中的光刻（光的衍射、波长）
- 量子通信简介（光子、偏振）
- 粒子加速器（洛伦兹力、回旋加速器）
- 光伏发电（光电效应、半导体）

### 能源与环保
- 风力发电（能量转换、效率）
- 水力发电（机械能守恒）
- 核电站（核裂变、质能方程）
- 太阳能电池（光电效应）
- 潮汐能（万有引力）

## 上海物理等级考题型分布参考

| 题型 | 题量 | 分值(参考) |
|------|------|-----------|
| 单选题 | 2-3/大题 | 2-3分/题 |
| 不定项选择 | 1-2/大题 | 3-4分/题 |
| 填空 | 1-2/大题 | 2-3分/空 |
| 计算/推导 | 1-2/大题 | 5-8分 |
| 简答/论述 | 1/大题 | 4-6分 |

## 上海物理等级考特点

1. **情景化命题**：每个大题目必须从真实物理情景出发
2. **推理计算并重**：不仅考计算，更考查物理推理过程
3. **开放性设问**：常有"分析原因""说明理由""评价方案"等简答题
4. **现代物理渗透**：近代物理（量子、相对论初步）占一定比例
5. **实验思想**：即使不是实验题，也常渗透实验设计思想
