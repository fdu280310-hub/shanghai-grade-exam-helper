# 上海等级考出题助手

将外地（全国卷、北京卷等）物理/化学题目一键转换为上海等级考风格的 Word 文档。

## 核心能力

1. **识别题目**：从图片（截图/拍照）或 PDF 中读取外地考试题目内容
2. **情景化改编**：为缺少情境的题目添加上海等级考标志性的"情景引入"（工业应用、科研成果、日常生活等真实场景）
3. **格式规范输出**：生成完全符合上海等级考排版规范的 .docx 文件

## 何时使用

**合适的场景**：
- 教师在网上看到外地好题（北京卷/全国卷/江苏卷等），想改编到周练/月考卷中
- 需要快速将一本外地教辅中的题目改成上海风格
- 需要为某个知识点补充等级考风格的练习题
- 出月考卷/周练卷时需要统一格式

**不合适的场景**：
- 需要生成整份完整试卷（本 skill 聚焦于单个大题或几个大题的改编）
- 小学/初中题目（上海等级考特指高中等级考）
- 语文/数学/英语/生物等其他科目（当前仅支持物理和化学）

## 工作流

### Step 1: 接收题目

从用户获取输入：
- **图片**：截图或拍照的外地题目。优先让用户提供清晰图片。
- **PDF**：包含题目的 PDF 文件。需要用户指定页码或大致位置。
- **纯文字**：用户直接复制粘贴的题目文本。

识别并确认：
- 科目（物理 or 化学）
- 题目结构（大题题干、子题、选项、图表描述等）
- 题目难度和考点

### Step 2: 改编为等级考风格

在改编之前，先阅读对应科目的参考文件：
- 化学 → `references/chemistry_standards.md`
- 物理 → `references/physics_standards.md`

改编规则参考 `references/question_adaptation_guide.md`。

#### 2.1 添加情景引入

如果原题没有上海等级考风格的情景引入，需要生成一个。参考 `references/scenario_examples.md` 获取灵感。

情景引入要求：
- 真实、科学、与题目内容自然衔接
- 来源于工业流程、科研前沿、日常生活、环保应用等
- 1-2 段，简洁明了
- 不能是"小明做实验"这类低幼化场景

#### 2.2 调整题型和编号

- 大题编号：一、二、三、……（chineseCountingThousand 格式）
- 子题编号：1．2．3．……（decimal 格式）
- 选项编号：A．B．C．D．
- 不定项选择：标注"（不定项）"，每小题1~2个正确选项
- 单选题：不特别标注，每小题1个正确选项

#### 2.3 确保难度匹配等级考

上海等级考特点：
- 重理解、重推理，计算量适中
- 强调知识在真实情境中的应用
- 常含开放性简答题（"简述原因"、"说明理由"）
- 混合题型：选择题 + 填空题 + 简答题 + 计算题

#### 2.4 展示改编结果给用户确认

在生成 docx 之前，先以 Markdown 格式展示改编后的题目内容和结构，让用户确认或修改。

### Step 3: 生成 Docx

确认后，构建 JSON 数据并调用生成脚本：

```bash
python "<SKILL_ROOT>/scripts/generate_docx.py" <data.json> <output.docx>
```

JSON 数据结构参见 `scripts/generate_docx.py` 中的 `SAMPLE_DATA`。

生成脚本的工作方式：
1. 将 `assets/template_unpacked/`（预配置好样式/字体的空白模板）复制到临时目录
2. 生成 `word/document.xml` 替换模板中的占位文件
3. 打包为标准 .docx（ZIP 格式）

### Step 4: 验证与交付

生成后检查：
- 字体层级是否正确（标题黑体16pt、大题标题黑体14pt、情景宋体10.5pt首行缩进……）
- 页面设置是否为 A4
- 行距是否正确
- 化学符号的上下标是否正常

交付文件给用户，支持根据反馈迭代修改。

## 格式规范速查

详细规范见 `references/formatting_standards.md`。

| 元素 | 字体 | 字号 | 对齐 | 行距 |
|------|------|------|------|------|
| 标题 | 黑体 | 16pt | 居中 | 360 |
| 大题标题 | 黑体 | 14pt | 居中 | 360 |
| 考生注意标题 | 黑体 | 10.5pt | 左 | 293 |
| 情景引入 | 宋体 | 10.5pt | 左(首行缩进480) | 293 |
| 小题题干 | 宋体 | 10.5pt | 左(悬挂缩进420) | 360 |
| 选项 | 宋体 | 10.5pt | 左 | 360 |
| 正文 | 宋体 | 10.5pt | 左(首行缩进480) | 293 |

页面：A4 (210mm×297mm)，页边距上/下 2.54cm，左/右 3.18cm。

## 科目规范要点

### 化学

- 文档开头须包含"相对原子质量"行（读 `references/chemistry_standards.md` 获取标准列表）
- 化学式中的数字用下标 `<w:vertAlign w:val="subscript"/>`
- 离子电荷用上标 `<w:vertAlign w:val="superscript"/>`
- 可逆反应用 ⇌ (U+21CC)
- 常见情景主题：工业合成、废水处理、电池技术、有机合成路线

### 物理

- 物理量用斜体，单位用正体
- 矢量用粗体或上方加箭头
- 图表需带坐标轴标注
- 有效数字通常保留2-3位
- 常见情景主题：交通工具、航空航天、体育运动、医疗设备

## 注意事项

1. **字体兼容**：模板使用 Windows 中文系统自带的 宋体、黑体、楷体。在 Mac/Linux 上编辑时可能会降级为系统默认字体，但打印效果通常不受影响。
2. **化学公式**：简单化学式用 Unicode + 上下标 XML 解决。极端复杂的结构式建议在 Word 中手动插入 ChemDraw 对象。
3. **图表处理**：原题中的图表描述保留在文字中。如需插入图片，在生成后在 Word 中手动操作。
4. **WPS 兼容**：模板来源于 WPS 创建的文档，在 WPS 中打开效果最佳，Microsoft Word 中也完全兼容。
